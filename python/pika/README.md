# pika example

## Installing

```bash
# download and install pyenv
curl -L https://github.com/pyenv/pyenv-installer/raw/master/bin/pyenv-installer | bash
# installing python 3.6.0
pyenv install 3.6.0
pyenv rehash
pyenv shell 3.6.0
pip install -r requirements.txt
```

## Executing

```bash
# run server
python srv.py
# run client
python rpc.py
```
